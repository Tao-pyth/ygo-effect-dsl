# Package 0.3.0 Release Checklist

Status: mandatory source milestone gate

## Reviewed source

- package version、README、roadmap、changelog、ADR 0014が`0.3.0`で一致する。
- Experiment `0.4`、Route DSL `0.1`、Replay/Search contractのversionを不要に変更していない。
- #108と#110を未完了のまま保持し、pruning defaultと一般deck evaluator claimを無効にしている。

## Verification

1. `python -m pytest`を完走する。
2. `python -m build`でwheelとsdistを作る。
3. `python -m ygo_effect_dsl.release_audit dist/*.whl dist/*.tar.gz`でpathとthird-party payload境界を検査する。
4. clean virtual environmentへwheelをinstallし、`--version`と`experiment-search --help`を実行する。
5. GitHub ActionsのTestsとBuild Windows exeを同一commitで成功させる。
6. `self-hosted-real-core-smoke-v1`の成功証跡とrunner deregistrationを確認する。

## Publication boundary

- PRをreview済み状態でmergeし、`main`を同期してからannotated `v0.3.0` tagを作る。
- tag workflowのTestsとBuild Windows exeが成功したことを#144へ記録する。
- GitHub Release、PyPI、一般配布は作らない。source tagは第三者assetまたはproject sourceの再配布承認を意味しない。
- tagは移動・再利用しない。失敗時は原因を修正し、新しいpatch versionで回復する。

実行run、commit SHA、artifact audit結果、tag確認の正本は[#144](https://github.com/Tao-pyth/ygo-effect-dsl/issues/144)に記録する。
